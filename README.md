# vasie.dev

Personal site. Astro + Tailwind v4, deployed on Cloudflare Workers.

## Commands

| Command            | Action                                 |
| :----------------- | :------------------------------------- |
| `npm install`      | Install dependencies                   |
| `npm run dev`      | Start dev server at `localhost:4321`   |
| `npm run build`    | Build to `./dist/`                     |
| `npm run preview`  | Build + preview locally                |
| `npm run deploy`   | Build + `wrangler deploy` to Workers   |

Node version pinned in `.nvmrc` (22.12.0).
